# React + Redux + webpack + Router

## 项目前景

需要搭建一个管理后台，有过`React Native`基础，故想来折腾一下这一套`大保健`，没想到需要注意的事情还是挺多的，特来记录一下。

## 项目运行
#### 由于涉及大量的 ES6/7 等新属性，建议`node`使用`6.7.0`以上

```
git clone https://github.com/5ibinbin/react-redux-webpack.git

npm install (安装依赖包)

npm start (运行本地开发环境)

npm run build (打包)

```
